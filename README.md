# Aeroshell - Modern SSH Client

## 🇺🇸 English

Aeroshell is a modern SSH client designed for developers and DevOps engineers.

Features:
- SSH / SFTP / VNC
- Multi-device management
- AI assistant

A great alternative to Xshell, MobaXterm and FinalShell.

Download:
https://termdev.com/download.html
